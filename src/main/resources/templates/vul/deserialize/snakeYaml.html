反序列化 - SnakeYaml 

  SnakeYAML 是 Java 常用 YAML 解析库。使用默认 Constructor 解析不可信 YAML 时，攻击者可能通过 !!全类名 指定任意 Java 类型，触发对象实例化、构造器、setter 或类加载链路。
  安全解析应使用 SafeConstructor 或受限 LoaderOptions，只允许标量、列表、Map 等基础数据结构进入业务层；不要把用户提交的 YAML 直接映射为任意 Java 对象。

漏洞场景：SnakeYaml

tips
默认 Yaml.load() 会根据 YAML 标签构造 Java 对象。示例 payload 使用项目内 Sqli 类，便于稳定观察“类型被实例化”的效果；真实攻击还可能结合 ScriptEngineManager、URLClassLoader、JNDI 或组件 gadget 链。
